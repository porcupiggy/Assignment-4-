# Assignment-4-
for this assignment i tried to go with something a little fun (but scary) so i chose volcanoes! 
the data for this assignment shows various volcanoes and uses a bar chart to show the tallest volcanoes in the world!

Data.
the data that is shown on this assignment is:
- name
- country
- region
- elevation
- lat
- long

i used a JSON dataset that i got from online, it has real information on the tallest volcanoes
i had issues with this assignment my browser for some reason did not like what was happening so it just constantly showed errors even when it shouldn't have been and after hours 
(and hours) of trying to solve it, i just decided to add a JSON file and put all the information in the file.

How to run it.
1. download the
   - index.html
   - script.js
   - volcanoes.json
2. open the index.html with live server (or just in your browser)
3. and the map and chart should automatically load!
